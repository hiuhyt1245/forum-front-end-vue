# forum-front-end-vue

## Features
- 使用者可於論壇內註冊帳號、登入登出帳號、評論餐廳、收藏或按讚餐廳及編輯個人資料等。
- 管理者可於後台新增、編輯、刪除餐廳資料。
- 應用前端框架 Vue.js 及 Bootstrap 套件，建造 SPA 網站。
- 透過 axios 串接 RESTful API 取得餐廳資料。
- 測試帳密：root@example.com / 12345678。
